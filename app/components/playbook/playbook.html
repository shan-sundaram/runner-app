Nodejs Server Playbook

by CenturyLink

Overview

Create a lightweight server that can handle a great number of simultaneous requests. Node.js is an open-source, cross-platform runtime environment for developing server-side web applications. It provides an event-driven architecture and a non-blocking I/O API designed to optimize an application's throughput and scalability for real-time web applications. Node.js contains a built-in library to allow applications to act as a web server without software such as Apache HTTP Server, Nginx or IIS.

Development Tasks

Gulp is used to manage the majority of the project's development tasks. Below outlines the most important ones; inspect Gulpfile.coffee to see them all.

Bower is used for client-side package management. Packages installed with Bower are stored in the source/vendor directory. The gulp and gulp build commands will automatically concatenate each package's "main" file into vendor.js.

  • To search for packages: bower search {package name}
  • To install a package: bower install {package name} --save
